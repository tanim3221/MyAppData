Back to main menu


2. Direct Unlock

  1. Insert into modem an un-acceptable SIM card;
  2. Connect modem to PC;
  3. Install necessary driver;
  4. Press "Unlock" button;
  5. Follow software instructions (promts).

Back to main menu


3. Repair IMEI

  1. Connect modem to PC with cable;
  2. Install necessary driver;
  3. Enter correct IMEI;
  4. Press "IMEI Repair" button;
  5. Follow software instructions (promts).

Back to main menu


4. Write firmware

  1. Install necessary drivers (drivers are located in "Drivers" folder);
  2. Select firmware file;
  3. Press "Write firmware" button;
  4. Follow software instructions (promts);

Back to main menu
