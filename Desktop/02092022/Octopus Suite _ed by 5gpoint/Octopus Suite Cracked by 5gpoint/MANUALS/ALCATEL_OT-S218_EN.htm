2. Calculate unlock codes

  1. Put your IMEI number in the IMEI cell;
  2. Press "Calculate" button;
  3. Follow software instructions (promts).

Note: To enter unlock code, please perform the following steps:

  1. Turn phone off and insert an un-acceptable SIM card;
  2. Turn phone on;
  3. Enter PIN code (if needed);
  4. When the message "Enter network key" appears, enter the Unlock Code;
  5. Restart phone.
