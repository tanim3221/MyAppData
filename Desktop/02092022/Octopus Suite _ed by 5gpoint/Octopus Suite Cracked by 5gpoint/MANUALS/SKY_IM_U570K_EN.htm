5. FAQ

Q: My phone not connected via USB. How can I configure my phone?
A: Perform following steps:

  1. Enter into "DEBUG MENU" in next way :
  2. Type ##000000# in dial window
  3. Enter like password phone code, default is "0000"
  4. Select "COMM.ENVIRONMENT"
  5. Set "DM Port" to USB, if you see baudrate - set it to 115200
  6. Set "DS Port" to USB, if you see baudrate - set it to 115200

Back to main menu


Note: The phone should be connected without SIM card!
