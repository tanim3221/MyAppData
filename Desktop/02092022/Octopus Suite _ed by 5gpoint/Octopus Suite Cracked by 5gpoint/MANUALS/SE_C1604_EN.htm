3. Write firmware

  1. Install necessary drivers (drivers are located in "Drivers" folder);
  2. Select firmware file;
  3. Press "Write firmware" button;
  4. Reconnect battery;
  5. Press "Back" button and connect phone with Micro USB cable directly to motherboard USB port (don't use USB hubs!).
  6. Follow software instructions (promts);

Back to main menu


4. Direct Unlock

  1. Select menu "Settings"->"Developer options"->"Enable USB debugging";
  2. Connect powered-on phone to PC with Micro USB cable;
  3. Install necessary driver;
  4. Press "Unlock" button;
  5. Follow software instructions (promts).

Back to main menu


5. Repair IMEI

  1. Select menu "Settings"->"Developer options"->"Enable USB debugging";
  2. Connect powered-on phone to PC with Micro USB cable;
  3. Install necessary driver;
  4. Enter correct IMEI;
  5. Press "Repair IMEI" button;
  6. Follow software instructions (promts).

Back to main menu
