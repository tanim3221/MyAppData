3. Write firmware

  1. Install necessary drivers (drivers are located in "Drivers" folder);
  2. Select firmware file;
  3. Press "Write firmware" button;
  4. Reconnect battery;
  5. Press "Back" button and connect phone with Micro USB cable directly to motherboard USB port (don't use USB hubs!).
  6. Follow software instructions (promts);

Back to main menu


4. Direct Unlock

  1. Install necessary drivers (drivers are located in "Drivers" folder);
  2. Press "Unlock" button;
  3. Disconnect battery;
  4. Attach Testpoint and connect phone with Micro USB cable directly to motherboard USB port (don't use USB hubs!).
  5. Follow software instructions (promts);

Back to main menu


5. Repair Phone

  1. Install necessary drivers (drivers are located in "Drivers" folder);
  2. Press "Repair Phone" button;
  3. Reconnect battery;
  4. Attach Testpoint and connect phone with Micro USB cable directly to motherboard USB port (don't use USB hubs!).
  5. Follow software instructions (promts);

Back to main menu


6. Repair IMEI

  1. Install necessary drivers (drivers are located in "Drivers" folder);
  2. Enter correct IMEI;
  3. Press "Repair IMEI" button;
  4. Disconnect battery;
  5. Attach Testpoint and connect phone with Micro USB cable directly to motherboard USB port (don't use USB hubs!).
  6. Follow software instructions (promts);

Back to main menu


7. Reset Pattern Lock

  1. Reconnect battery;
  2. Press "Reset Pattern lock" button;
  3. Attach testpoint, according to the picture;
  4. Connect USB cable;
  5. Install necessary drivers (drivers are located in "Drivers" folder);
  6. Follow software instructions (promts).

Back to main menu
