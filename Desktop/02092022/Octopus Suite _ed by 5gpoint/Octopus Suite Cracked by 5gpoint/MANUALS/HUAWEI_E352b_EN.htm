2. Direct Unlock

  1. Insert into modem an un-acceptable SIM card;
  2. Connect modem to PC;
  3. Install necessary driver;
  4. Press "Unlock" button;
  5. Follow software instructions (promts).

Back to main menu


3. Read unlock codes

  1. Connect modem to PC;
  2. Install necessary driver;
  3. Press "Read codes" button;
  4. Follow software instructions (promts).

Note: To enter unlock code, please perform the following steps:

  1. Insert into Your modem unsupported SIM card;
  2. Connect modem to PC;
  3. Start Your Connection manager;
  4. Then modem will recognize unsupported SIM card and ask You to enter Unlock Code;
  5. After entering the Unlock Code Your modem will be unlocked.

Back to main menu


4. Calculate unlock codes

  1. Put your IMEI number in the IMEI cell;
  2. Press "Calculate" button;
  3. Follow software instructions (promts).

Back to main menu
