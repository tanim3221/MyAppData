2. Read unlock codes

  1. Connect modem to PC;
  2. Install necessary driver;
  3. Press "Read codes" button;
  4. Follow software instructions (promts).

Note: The modem should be connected without SIM card!

Note: To enter unlock code, please perform the following steps:

  1. Insert into Your modem unsupported SIM card;
  2. Connect modem to PC;
  3. Start Your Connection manager;
  4. Then modem will recognize unsupported SIM card and ask You to enter Unlock Code;
  5. After entering the Unlock Code Your modem will be unlocked.

Back to main menu


3. Write firmware

  1. Install necessary drivers (drivers are located in "Drivers" folder);
  2. Select firmware file;
  3. Press "Write firmware" button;
  4. Follow software instructions (promts);

Back to main menu


4. Download mode

  1. Reconnect battery;
  2. Press and hold "Connect" and "Power" button until appear "Update?" message;
  3. Press "Connect" button;
  4. Connect USB cable;
  5. Install necessary drivers.

Back to main menu
